# Creacion de Projecto SkyCast 

## En este proyecto lo que se busca realizar es una aplicacion de pronostico del clima, utilizando la API de OpenWeatherMap, para esto se utilizara el lenguaje de programacion Python y el framework de desarrollo web Flask.  
## Con esta aplicacion ayudaremos a los usuarios de la empresa de transporte aereo a conocer el pronostico del clima en los destinos a los que se dirigen, para asi poder tomar decisiones informadas sobre sus viajes.
